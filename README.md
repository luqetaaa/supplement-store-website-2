# Supplement Store Website

Projeto front-end de loja virtual de suplementos desenvolvido com HTML, CSS e JavaScript.

## 🚀 Sobre o projeto

Este projeto simula um e-commerce moderno de suplementos, com foco em design responsivo, navegação intuitiva e experiência visual profissional.

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📱 Funcionalidades

- Layout responsivo para desktop e mobile
- Catálogo de produtos
- Interface moderna
- Navegação fluida
- Estrutura organizada em pastas

## 🎯 Objetivo

Praticar desenvolvimento front-end e criação de interfaces modernas para web.
